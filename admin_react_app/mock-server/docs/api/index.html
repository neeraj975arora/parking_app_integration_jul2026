🚗 Parking Admin API

Comprehensive Mock Server for Parking Management System

📋 API Overview

This mock server provides realistic REST API endpoints for testing the Parking Admin Dashboard React application. It includes comprehensive authentication, session management, financial operations, and monitoring capabilities.

🔐 Authentication

  • JWT-based authentication
  • Role-based access control
  • Super Admin & Admin roles
  • Token refresh mechanism

👥 Admin Management

  • Create and manage admins
  • Parking lot assignments
  • Role-based permissions
  • Admin activity tracking

🚙 Session Management

  • Vehicle check-in/check-out
  • Real-time session tracking
  • Payment processing
  • Session analytics

💰 Financial Operations

  • Daily closure management
  • Payment tracking
  • Revenue calculations
  • Financial reporting

🔑 Authentication Required

Most endpoints require authentication. Use the login endpoint to obtain a JWT token:

POST /auth/login
{
  "user_email": "admin@parking.com",
  "user_password": "Admin@123",
  "role": "admin"
}

Then include the token in the Authorization header:

Authorization: Bearer <your-jwt-token>
